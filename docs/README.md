# Look Over Here — Project Docs

This folder contains the current source-of-truth design documents for **Look Over Here**.

## Current Direction

**Look Over Here is a shared-discovery exploration game about visiting impossible national parks from another universe.**

Players give the world a few strange instructions, then explore scenic low-poly places with huge views, readable trails, distant landmarks, moving curiosities, and surprising systemic reactions.

The old “N64” direction is no longer the main target. N64 remains useful only as a reference for readability, simple forms, strong silhouettes, and charm.

The current visual target is:

> **Stylized low-poly national park adventure with modern scenic draw distance, breathtaking landscape views, saturated dreamlike color, readable trails, high climbs, and strange distant landmarks.**

## Files

- `north-star.md` — project vision, core fantasy, emotional target, loop, and boundaries.
- `scenic-low-poly-art-direction.md` — current visual direction: impossible national parks, modern draw distance, vistas, atmosphere, shaders, and low-poly readability.
- `world-biomes.md` — first impossible national park biome concepts for the future WHERE? wheel.
- `generator-design.md` — three-wheel input system and slot-based generation design.
- `prototype-plan.md` — revised prototype direction focused on Rose Canyon Highlands / scenic vista exploration.
- `tech-direction.md` — browser/Three.js prototype direction updated for modern scenic low-poly needs.
- `n64-art-reference.md` — legacy N64-inspired lessons to preserve, without treating N64 as the visual target.

# Look Over Here — Project Docs

This folder contains early design documents for **Look Over Here**, a small co-op procedural exploration game about shared discovery.

## Files

- `codex-instructions.md` — project-specific Codex workflow, scope, verification, and completion rules.
- `north-star.md` — project vision, player fantasy, core loop, design pillars, and prototype target.
- `n64-art-direction.md` — visual target, N64-inspired art pillars, readability rules, and first biome direction.
- `prototype-plan.md` — first playable prototype scope, procedural limits, content kit, and success criteria.
- `tech-direction.md` — recommended browser/Three.js prototype direction, technical boundaries, and future co-op cautions.
